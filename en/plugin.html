Tiefsee Plugin List
QuickLook
Select a file on the desktop or in a folder and then press and hold the spacebar to preview the file
Download

GitHub：
https://github.com/hbl917070/Tiefsee.QuickLook
monaco-editor
Let Tiefsee use monaco-editor to load text files. Commonly used for reading and editing code
Download


GitHub：
https://github.com/microsoft/monaco-editor
WebViewer
Let Tiefsee support "doc, docx, ppt, pptx"
Download
Click "Download" to enter the official website's download center.
It is recommended to download version 8.12, as other newer versions have not been tested for compatibility with Tiefsee.

Web → Older Versions → Version 8.12

※ After unzipping, if the folder name is "WebViewer-8.12.0", change it to "WebViewer".



WebViewer is a product of PDFTron, and their official website provides a free trial. If you need a license for PDFTron, please visit:
https://www.pdftron.com
Plugin Installation Method
The plugins downloaded here are ZIP files. Unzip and place them in the 'Plugin' folder
Then restart Tiefsee.

You can open the 'Plugin' folder from here
