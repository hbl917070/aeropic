Tiefsee Plugin List
QuickLook
Select a file on the desktop or in a folder and then press and hold the spacebar to preview the file
Download

GitHub：
https://github.com/hbl917070/Tiefsee.QuickLook
monaco-editor
Let Tiefsee use monaco-editor to load text files. Commonly used for reading and editing code
Download


GitHub：
https://github.com/microsoft/monaco-editor
WebViewer
Let Tiefsee support "doc, docx, ppt, pptx"
Download
Click 'Download' to go to the download center on the official website,
Web → SDK Downloads → WebViewer



WebViewer is a product of PDFTron and the official website offers a free trial.
If you need a PDFTron license, please visit: https://www.pdftron.com
Plugin Installation Method
The plugins downloaded here are ZIP files. Unzip and place them in the 'Plugin' folder
Then restart Tiefsee.

You can open the 'Plugin' folder from here
