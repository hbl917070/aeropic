Tiefsee 是一款適用於 Windows 的開源圖片檢視器，具有強大的功能和易用性。
該專案採用 .NET 7 構建，以 C#、TypeScript、SCSS 和 EJS 進行開發，
整個使用者界面都是使用 WebView2 渲染的，提供了流暢的使用體驗。

運行最低需求：64位元的 Windows 10

商店版 與 免安裝版 有什麼差異？

  • 商店版 與 免安裝版 功能上沒有差異
  • 商店版 可以透過商店來進行更新，且刪除 Tiefsee 後不會留下任何資料
  • 免安裝版 需要安裝 .NET Desktop Runtime 7 (x64) 才能運行
  • 免安裝版 可以使用「便攜模式」 ，在 Tiefsee.exe 旁邊新建一個 portableMode 資料夾，資料就都會儲存在裡面
沒有特別需求的話，推薦使用商店版

為什麼 商店版 比 免安裝版 還要大？

商店版 在編譯的時候會把 .NET 的執行環境整個打包進程式裡面，微軟目前尚未提出有效的解決方案

Tiefsee 是否會收集使用者資料？

Tiefsee 不會收集使用者的任何資料，
不過使用「搜圖」時，Tiefsee 會先把圖片上傳至 暫存圖片的伺服器， 然後才上傳至第三方的圖片搜尋引擎，
在伺服器上的圖片副本，會在60秒後永久刪除。

啟動程式時顯示「WebView2 must be installed to run this application」

可能的原因如下：
  • 電腦尚未安裝 WebView2：「點此」進行下載
  • 已經安裝 WebView2 但安裝失敗：移除舊的 WebView2 後，對「WebView2 的安裝檔」右鍵→系統管理員身份執行
  • WebView2 初始化錯誤：避免在 Tiefsee 的路徑裡面包含中文日文韓文之類的特殊字元

使用毛玻璃(AERO、Acrylic)視窗效果後產生異常

毛玻璃視窗效果並非 Windows 正式公開的 API，這項功能在某些裝置上可能存在 BUG，或是無法使用。常見的問題為：
  • 模糊區域溢出到視窗外
  • Tiefsee 的視窗在移動時嚴重延遲
