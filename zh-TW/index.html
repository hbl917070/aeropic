Tiefsee

一款適用於 Windows 的開源圖片檢視器，具有強大的功能和易用性
已推出 4.2.1 版
從商店獲取 Tiefsee
下載免安裝版 更新說明 查看舊版 3.0.2
視窗主題
視窗效果：
半透明視窗、AERO、Acrylic

自訂選項：
背景色, 文字顏色, 圖示顏色, 圖示粗細, 文字粗細, 視窗圓角
檔案預覽面板
查看目前資料夾中的所有圖片，方便切換圖片和管理圖片
資料夾預覽面板
查看目前資料夾周圍的同級資料夾，並預覽資料夾內的圖片，方便切換資料夾和管理資料夾
詳細資料面板
讀取圖片內的附加資訊：
  • EXIF 資訊，並以 Google Map 顯示拍攝地點
  • AI 繪圖參數，支援
    • A1111 (Stable Diffusion web UI)
    • NovelAI
    • ComfyUI
    • InvokeAI
    • StableSwarmUI
大量瀏覽模式
以垂直捲動的方式查看多張圖片
  • 可讓圖片垂直排列或雙欄排列，適合用於看漫畫
  • 使用瀑布流自動排列圖片，同時查看多張圖片
快速拖曳
直接從 Tiefsee 將圖片拖曳至其他程式進行開啟或上傳
開啟網頁圖片
可從 瀏覽器 或 Discord 把圖片拖曳至 Tiefsee 來開啟
支援多種格式
支援超過50種圖片格式：
svg、webp、webm、psd、clip、heic、avif、qoi ...
快速啟動
所有的 Tiefsee 共用同一個執行個體。只要 Tiefsee 的視窗尚未全部關閉，就能快速啟動 Tiefsee
線上搜圖
支援：
sauceNAO, Yandex, Ascii2d, Google, Google Lens, Bing
QuickLook
在桌面或資料夾選中任一檔案，然後長按空白鍵，即可預覽檔案
其他功能
Markdown 編輯器, 文字檔編輯器,
PDF 閱讀器, docx 閱讀器, pptx 閱讀器
