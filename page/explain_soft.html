說明
AeroPic 是一款圖片檢視軟體，可以將圖片或資料夾拖曳至程式中開啟，或設定成系統預設開啟圖片的程式（設定教學）。

程式開啟圖片後，會自動讀取該資料夾底下所有的圖片，供快速切換瀏覽。支援格式：
JPG、PNG、GIF、BMP、TIF、APNG、SVG、PSD、Pixiv動圖（下載教學）

滑鼠滾輪捲動可控制圖片縮放
圖片小於視窗時，拖曳=拖曳視窗
圖片大於視窗時，拖曳=移動圖片位置

備註：
  • 只有windows10可以使用透明玻璃特效
  • 如果無法執行：請安裝 Microsoft .NET Framework 4.5 ，或更高版本
  • 如果如法順利瀏覽GIF 或 無法開啟大量瀏覽模式：請把IE更新到版本 IE11
  • 如果無法開啟PSD：右鍵→設定，把【PSD使用內建解碼器】打勾
