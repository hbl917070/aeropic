Aero 是 Windows 7 作業系統採用的一種透明玻璃界面
AeroPic 圖片檢視器 採用100%的透明玻璃界面，可用來取代windows內建的圖片檢視器，支持多種圖片格式（JPG、PNG、GIF、BMP、TIF、ICO、PSD、SVG、APNG、webP、RAW 等...）
